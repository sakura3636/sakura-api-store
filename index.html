🌸🌸Sakura API Store 公式サイト🌸🌸

Sakura API Store は、KabuステーションAPIを題材とした Pythonプログラミング学習教材を提供する公式ショップです。

当ショップが提供する商品

  • Pythonによる自動売買プログラムの入門教材
  • 自動売買のシステム構造を理解するための応用教材
  • 本格的な自動売買システム開発のための上級者向け教材
  • 追加モジュール・オプション教材等の提供
商品一覧を見る

動作環境・利用条件（全て必須）

  • Windows 10または Windows 11 のインターネット接続可能なパソコン
  • Intel Core i5または同等以上のCPU、8GB以上のRAM、20GB以上の空ディスク
  • テキストエディタでテキストファイルを編集できること（メモ帳でも可）
  • Anaconda3 のインストール（教材にマニュアル同梱）
  • KabuステーションのAPIパスワード取得（マニュアル同梱）

スターターキットについて（2026年3月発売予定）

本商品は、日経225マイクロ先物を題材として、Pythonによる 自動売買プログラムの基本構造を理解するための
初心者向けプログラミング学習用教材キットです。 将来、本格的なシステムを開発したい人が手軽に始められるように、
最小限のシステム構成キットを提供します。
